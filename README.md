## 프로젝트 개요
리얼타임테크에서 ETRI의 DB 데이터를 웹을 통해 확인하고 실시간 사용량을 차트로 보는 웹이 필요하다는 요청으로 datatables.js, chart.js, dtree.js 오픈소스를 활용해 개발해본 프로젝트입니다.
